# Sponsor logos

Drop SVG (preferred) or PNG logo files here, then reference them by
filename in `web/src/app/(auth)/layout.tsx` under the `SPONSORS` array.

Recommended:
- Single-color flat SVGs that work on a light background
- 24–32px target height
- Filenames lowercase, hyphen-separated (e.g. `damm.svg`, `engineering-hub.svg`)

Until logos are dropped here, the marketing panel renders text wordmarks
as a graceful placeholder.
